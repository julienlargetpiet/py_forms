# Functions

## pyramids

`pyramids(nb_block, block_shape="#")`

### Args

- `nb_block` is the number of blocks used to build the pyramid
- `block_shape` is the shape of the block used

### Examples

`pyramids(36, "3")
        3
       3 3
      3 3 3
     3 3 3 3
    3 3 3 3 3
   3 3 3 3 3 3
  3 3 3 3 3 3 3

level number:  8
